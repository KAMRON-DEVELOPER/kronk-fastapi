Account Deletion Information

If you decide to delete your account, all associated data will be permanently removed. This includes your profile, feeds, likes, bookmarks, messages, and any other personal content.

Deleting via the App

If you're using the mobile app, you can delete your account by:

  • Opening the Settings screen
  • Scrolling to the bottom
  • Tapping the red Delete Account button

Deleting via API (Without App)

You can also delete your account using our API if you prefer not to use the app:

  1. Send a POST request to https://api.kronk.uz/api/v1/users/login with your username and password in JSON format.
  2. The response will contain a tokens object with your access token.
  3. Use the access token in the Authorization header as a Bearer token and send a DELETE request to:

https://api.kronk.uz/api/v1/users/profile/delete

Example header:

Authorization: Bearer <your-access-token>
